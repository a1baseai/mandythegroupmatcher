## Email Details

### **Visual Design:**
- **Color Scheme:** Harvard Crimson (#A51C30) for headers and button
- **Font:** Arial, sans-serif
- **Layout:** Centered, max-width 600px
- **Button:** Red button with white text, rounded corners

### **Content Sections:**
1. **Header:** "🎉 You've Been Matched!"
2. **Greeting:** Personalized with group name
3. **Match Announcement:** Shows matched group name
4. **Compatibility Score:** Displayed if available (e.g., "87%")
5. **Call-to-Action:** Large button to join group chat
6. **Link Fallback:** Plain text link below button
7. **Next Steps:** Explains what happens after joining
8. **Signature:** "Mandy the Matchmaker"

### **Personalization:**
- Each group receives a personalized email
- Subject line includes the OTHER group's name
- Body is customized for each group
- Shareable link is the same for both groups

### **Example for Group 2:**

**Subject:** `🎉 You've been matched with The Justice League!`

**Body:** Same format but with names swapped:
- "Hi The Avengers,"
- "You've been matched with **The Justice League**!"
- Same compatibility score and link
